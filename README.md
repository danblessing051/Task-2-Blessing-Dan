---

 📌 Overview-----
Performed a full Exploratory Data Analysis on a 
real-world e-commerce orders dataset containing 
1,200 records across 14 columns covering January 
2023 to June 2025. The goal was to uncover hidden 
patterns, trends, outliers, and business insights 
from the data.

---

 🎯 Goal-----
Analyze a dataset to understand patterns, trends, 
and distributions.

---

 ✅ Key Requirements Completed
- Calculated basic statistics (mean, median, count)
- Identified trends and outliers
- Summarized key observations
- Built visualizations using Python and Excel

---

 📦 Dataset

| Detail | Info |
|--------|------|
| Records | 1,200 orders |
| Columns | 14 |
| Period | Jan 2023 – Jun 2025 |
| Type | E-Commerce Orders |

---

 🔑 Key Findings

| Finding | Result |
|---------|--------|
| Mean TotalPrice | $1,054 |
| Median TotalPrice | $824 |
| Distribution | Right-Skewed |
| Cancellation + Return Rate | 41.4% ⚠ |
| Top Product by Revenue | Chair ($195,620) |
| Top Referral Source | Instagram (21.6%) |
| Strongest Correlation | UnitPrice vs TotalPrice (0.72) |
| Outliers in TotalPrice | 8 detected via IQR method |

---

 📈 Analysis Performed
- Missing values check
- Descriptive statistics (mean, median, std, min, max)
- Order status distribution
- Revenue by product
- Monthly revenue trend
- Outlier detection using IQR method
- Correlation heatmap

 ▶ How to Run

1. Open Google Colab at colab.research.google.com
2. Click File → Upload Notebook
3. Upload "Eda.ipynb" from this repository
4. Click the folder icon on the left panel
5. Upload "Dataset for data Analytics.csv" to the Colab session
6. Run each cell one by one using Shift + Enter
7. All charts will appear below each cell

---

 💡 Key Insight-------
TotalPrice is right-skewed meaning the Mean ($1,054) 
is higher than the Median ($824). A 41.4% combined 
cancellation and return rate is a critical business 
problem that requires urgent investigation.
